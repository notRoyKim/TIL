# 마크다운 문법

## 1. header

+ 문서 제목
 
```
this is title
=============
```

결과
> this is title
> =============

+ 문서 부제

```
this is title
-------------
```

결과
> this is title
> -------------

+ 글머리 1 ~ 6


```
# this is an H1
## this is an H1
### this is an H1
#### this is an H1
##### this is an H1
###### this is an H1
```
결과
> # this is an H1
> ## this is an H1
> ### this is an H1
> #### this is an H1
> ##### this is an H1
> ###### this is an H1
